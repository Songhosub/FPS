# FPS 게임

### 프로젝트 소개
FPS게임의 기본적인 구성요소들을 가지고 있는 게임입니다.

### 프로젝트 동기
1) 이전까지 배운 다양한 기능들을 구현
2) 3D게임의 제작 경험
3) 시네머신을 이용한 프로젝트 제작

### 프로젝트 언어 및 환경
  프로그래밍 언어: C#  
  게임엔진 : Unity
