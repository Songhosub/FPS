# FPS 게임

### 프로젝트 소개
전략시뮬레이션의 유닛들을 RPG처럼 육성하는 기본적인 SRPG요소들을 가진 게임입니다.

### 프로젝트 동기
1) 이전까지 배운 다양한 기능들을 구현
2) Astar를 비롯한 탐색 알고리즘의 구현및활용
3) 처음으로 일정규모 이상의 개인 프로젝트 제작

### 프로젝트 언어 및 환경
  프로그래밍 언어: C#  
  게임엔진 : Unity
